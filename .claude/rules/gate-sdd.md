# Gate 2: SDD → Feature 开发门禁

> ARCHITECTURE.md 必须满足以下条件才能开始 Feature 开发。

## Checklist

- [ ] 总体架构图已定义（进程模型、通信路径）
- [ ] 所有进程模型已定义（Control Plane / Worker / Bot Worker 职责边界）
- [ ] 通信协议已定义（gRPC proto 或至少接口描述、WS 消息格式）
- [ ] 数据库模型已定义（核心表结构、字段类型、索引）
- [ ] 前端架构已定义（技术栈、目录结构、路由策略）
- [ ] 所有架构决策已有 ADR 记录
- [ ] ARCHITECTURE.md 和 PRD 无矛盾

## 未通过处理

补充 ARCHITECTURE.md 对应项，不得开始编码。

## 检查方式

按此 checklist 逐项核对。
