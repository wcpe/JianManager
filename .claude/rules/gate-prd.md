# Gate 1: PRD → SDD 门禁

> PRD 必须满足以下条件才能进入 SDD（系统设计）阶段。

## Checklist

- [ ] 所有 P0 FR 已定义，状态为 `📋 计划` 或更高
- [ ] 每个 FR 有明确的验收标准（可测试的 checklist）
- [ ] FR 之间无逻辑矛盾（A 功能不会否定 B 功能的前提）
- [ ] 非功能需求已定义（性能目标、安全要求、部署约束）
- [ ] V1 范围边界已明确（做什么 / 不做什么，清晰列出）
- [ ] 每个 FR 标注了优先级（P0/P1/P2）

## 未通过处理

补充 PRD 对应项，不得进入 SDD 阶段。

## 检查方式

按此 checklist 逐项核对。
